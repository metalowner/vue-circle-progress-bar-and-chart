# Vue circle loader and pie chart

Includes functional, styled and reusable components made for Vue 3 Composition API.

## Circel progress bar
Key features:
- Responsive
- Animated
- Can be made into a dashboard (not a full circle)

## Pie chart
Key features:
- Responsive
- Editable (add, update, delete)

## Nav bar
Key features:
- Responsive
- Animated
- Folds on scrollDown

### Depends on
- Vue Charts js
- Vue color

### Started as a test project
Yet I am willing to polish the components to perfection and use when needed.
